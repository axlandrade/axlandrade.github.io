Áudio computacional

MusicaPython

Produção técnica registrada no Lattes em 2020, criada a partir da interseção entre matemática, áudio e programação.

Ver repositório Outros projetos

O que é

O MusicaPython aproxima conceitos de séries de Fourier, frequências e ondas sonoras de uma aplicação real. A proposta é transformar dados musicais em áudio sintetizado, permitindo experimentar com instrumentos, notas e processamento de arquivos MIDI.

Registro e funcionalidades

  • Conversão de arquivos MIDI para WAV.
  • Suporte a múltiplos instrumentos e trilhas.
  • Interface web construída com Flask.
  • Ambiente Dockerizado para facilitar execução.
  • Produção técnica em coautoria com Mauro Benayon.

Tecnologias

Python Flask Docker MIDI WAV
